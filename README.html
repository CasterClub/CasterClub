About CasterClub

CasterClub is the rebirth of one of the first internet radio host platforms. From SHOUTcast/ICEcast to modern AI-enabled tools, CasterClub provides the ecosystem for hobbyists and professionals alike to manage, schedule, and broadcast live or on-demand audio/video streams.

Legacy Services (2002 - 2012)

  • SHOUTcast (MP3, MP3PRO, AAC/PLUS, Video)
  • Icecast2 (NSV/Video, MP3, Ogg Vorbis)
  • Storage Plans (from 50mb to 700mb)
  • Community Forums, Wiki, and 99.9% SLA Uptime

Modern CasterClub Projects

  • CasterClub Platform: Fully modernized web broadcasting and podcasting dashboard
  • CastIt: Web-based live streaming + AI-driven automation tools
  • AutoDJ Agent: NLP-based AI DJ module that handles dynamic content curation
  • Community XP: Windows XP-style forums, support boards, and user engagement gamification
  • New Icecast2 Fork (Golang): Built for low-latency, AI integration, OAuth2 support
